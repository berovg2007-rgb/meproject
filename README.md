# Task Manager Pro
**Автор:** Беров Григорий

## Описание
Приложение для управления списком задач с возможностью фильтрации по категориям и сохранения данных в JSON.

## Функционал
- Добавление задач с валидацией.
- Удаление выбранных задач.
- Фильтрация по категориям (Работа, Учеба и др.).
- Автоматическое сохранение в `tasks.json`.
- Кнопка "Удача дня" (использование библиотеки random).

## Инструкция по запуску
1. Установите Python 3.x.
2. Склонируйте репозиторий: `git clone [ссылка]`.
3. Запустите приложение: `python main.py`.
4. Для запуска тестов: `python tests.py`.

## Примеры использования
Введите "Изучить Git" в поле названия, выберите "Учеба" и нажмите "Добавить". Задача появится в списке и сохранится в файл.
