# Movie Library
**Автор:** Беров Григорий

## Описание
Приложение для хранения личной кинотеки. Позволяет добавлять фильмы, указывать их жанр, год и рейтинг. Данные сохраняются в JSON-файл. Реализована фильтрация по жанрам и годам выпуска.

## Инструкция по запуску
1. Установите Python 3.x.
2. Запустите файл командой: `python main.py`.

## Примеры использования
Введите "Начало", выберите "Фантастика", год "2010", рейтинг "8.8". Нажмите добавить. Чтобы найти все фильмы 2010 года, введите год в поле фильтра и нажмите "Применить".
